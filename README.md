# Timeline — Timeline Embed Tool

Display timelines on any website with ease.

## What It Is

Timeline is a SaaS embed tool designed for closed-source site builders like SITE123.

On platforms like SITE123, Wix, or Squarespace where you can't install plugins, just add a simple `<script>` embed code to display timeline content — no modifications to the platform itself required.

## Use Cases

- Display key dates on an event page
- Show version history on a product page
- Present company milestones on an "About Us" page

## How It Works

1. Create a timeline (title + event dates) in the Timeline admin dashboard
2. Copy the generated `<script>` embed code
3. Paste the code into any page on your target website
4. Visitors see the rendered timeline when they load that page

## Key Features

- Multiple timeline management
- Supports date, year, and month precision
- Two embed modes: Pure Embed / URL Check (loads only on specified page)
- Responsive layout for desktop and mobile
- No plugins needed on the target website

## Quick Start

1. Register an account
2. Create your first timeline
3. Copy the embed code to your target website

## Tech Stack

- Backend: Express.js + sql.js (SQLite in-memory database)
- Frontend: Plain JavaScript SPA (hash routing)
- Deployment: PM2 + Nginx
